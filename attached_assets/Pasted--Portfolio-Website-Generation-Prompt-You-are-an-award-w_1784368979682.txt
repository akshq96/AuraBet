# Portfolio Website Generation Prompt

You are an award-winning frontend engineer and creative designer. Your task is to build a portfolio website that looks like it could win a hackathon demo award. Do NOT generate a generic portfolio. Avoid templates, boring cards, and standard layouts.

## Goal

Build a portfolio that instantly makes recruiters, founders, hackathon judges, and engineers think:

> "This developer cares about craft."

The website should feel premium, modern, interactive, and memorable while remaining professional.

---

## Tech Stack

* Next.js 15 (App Router)
* React
* TypeScript
* Tailwind CSS
* Framer Motion
* GSAP (only where necessary)
* Three.js / React Three Fiber (light usage)
* Lucide Icons
* Responsive
* SEO optimized
* Dark mode by default
* Fast loading
* Accessible
* Deployable on Vercel

---

## Overall Design Direction

The design should NOT resemble a normal portfolio.

Avoid:

* Hero + About + Projects + Contact stacked vertically
* Plain white cards
* Bootstrap style layouts
* Generic gradients
* Copy-paste developer portfolios
* Apple clone
* Linear clone
* Framer template clone

Instead create something cinematic.

Theme:

"A software engineer's digital workspace."

Imagine entering someone's futuristic workstation where every section feels alive.

The page should feel like an operating system mixed with a developer dashboard.

---

## Visual Style

Use:

* Glassmorphism
* Soft neon glows
* Subtle noise texture
* Animated gradients
* Aurora backgrounds
* Floating particles
* Depth
* Smooth shadows
* Dynamic lighting
* Rounded corners
* Premium typography

Colors:

Mostly dark.

Accent colors:

* Electric Blue
* Purple
* Cyan

Keep colors minimal.

---

## Animations

Everything should feel alive.

Examples:

* Mouse parallax
* Cursor glow
* Magnetic buttons
* Smooth page transitions
* Text reveal animations
* Scroll-driven animations
* Sections fade naturally
* Cards lift on hover
* Buttons ripple
* Gradient movement
* Background reacts slightly to mouse

Animations should never feel distracting.

Everything should be buttery smooth.

Target 60 FPS.
